# DatePicker 日期选择器

用于选择或输入日期

# 选择日

以「日」为基本单位，基础的日期选择控件

基本单位由type属性指定。快捷选项需配置picker-options对象中的shortcuts，禁用日期通过 disabledDate 设置，传入函数

<dy-date-picker v-model="value1"></dy-date-picker>
<template>
  <dy-date-picker v-model="value1"></dy-date-picker>
</template>
<script>
  export default {
    data() {
      return {
        value1: new Date(),
      };
    },
  };
</script>
