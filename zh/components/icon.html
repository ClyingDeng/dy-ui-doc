# Icon 图标

提供了一套常用的图标集合。

# 使用方法

直接通过设置类名为 dy-icon-iconName 来使用即可。例如：

<dy-icon class="dy-icon-edit"></dy-icon>
<dy-icon class="dy-icon-share"></dy-icon>
<dy-icon class="dy-icon-delete"></dy-icon>
<dy-button type="primary" icon="dy-icon-search">搜索</dy-button>
